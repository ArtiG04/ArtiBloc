Descripción general

PyroPrev es un dispositivo pensado para la detección preventiva de incendios forestales, usando sensores de humedad, temperatura y humo, y comunicación LoRa para formar redes de detección.

1 · Descripción del proyecto+

PyroPrev detecta de forma preventiva incendios forestales mediante sensores de humedad, temperatura y humo. Los dispositivos se comunican entre ellos por LoRa para crear una red que pueda retransmitir la alarma hasta un punto de control con personas cualificadas.

Cada dispositivo tiene un identificador y su ubicación GPS, y en caso de detección envía mediciones y coordenadas. La red repite la senal hasta llegar a las autoridades.

2 · Problemas potenciales+

Problema principal 1: Suministro de energía — opciones: panel solar + baterías (mantenimiento, visibilidad), baterías (coste/recambio), investigación en generación bioeléctrica desde el propio árbol.

Problema 2: Alcance de detección — la altura y orientación de los sensores afectan el rango efectivo.

Problema 3: Alcance de transmisión LoRa — antena en copa mejora el alcance; doble antena y antenas de ganancia podrían ayudar.

3 · Mejoras y soluciones futuras+

Investigar suministro por electrodos en el árbol, usar sensores más sensibles o distribuir sensores por ramas (compromiso tamano/fiabilidad). Anadir anemómetro para medir viento (clave en expansión de incendios). Crear un punto de control con mapa interactivo para visualizar dispositivos en tiempo real.

4 · Piezas y presupuestos+

Resumen de componentes (ejemplo):

  • ESP32 WROOM-32D
  • Módulo LoRa SX1278 (433MHz)
  • Sensor temp/humedad AM2302 / SHT40
  • Sensor humo MQ-2
  • Baterías Li-ion, panel solar, electrodos Cu/Zn (I+D)

5 · Pines y conexiones (resumen)+

Conexiones clave:

LoRa SX1278 (SPI):
  MOSI -> GPIO23
  MISO -> GPIO19
  SCK  -> GPIO18
  NSS  -> GPIO5
  RESET-> GPIO14
  DIO0 -> GPIO26

MQ-2:
  AOUT -> GPIO34 (ADC)

DHT/DHT11:
  DATA -> GPIO4 (pull-up)

GPS NEO-6M (opcional):
  RX/TX -> Serial2 (GPIO16/17)

Botón: GPIO13 (INPUT_PULLUP)
